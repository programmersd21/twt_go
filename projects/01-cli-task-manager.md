# 01. CLI Task Manager (Beginner)

**Concepts:** `Structs`, `Slices`, `Maps`, `Methods`, `Error Handling`, `File I/O`.

### Description
Build a command-line tool to manage your daily tasks.
- **Features:** Add, list, mark as done, and delete tasks.
- **Goal:** Practice data structures and basic logic.
- **Bonus:** Persistence—save and load tasks from a `JSON` or `CSV` file.

---

## How to Get Started
1. Create a new directory for your project (e.g., `projects/task-manager`).
2. Initialize a go module: `go mod init task-manager`.
3. Start coding! Refer back to the `src/` and `docs/` folders for help.
